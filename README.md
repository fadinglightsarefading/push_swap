**Description**\
This project involved making a program that displays the fewest possible moves to be made using a
given algorithm and two stacks (`a` and `b`). The moves (operations) include swapping the first two values in a stack (`sa`, `sb` and `ss`); rotating them so that the `int`s move upwards towards the
head (`ra`, `rb` and `rr`); rotating them contrariwise (`rra`, `rrb` and `rrr`); and pushing a head
value to either stack from the other (`pa` and `pb`).\
I used the Turk algorithm.

**Instructions**\
Use `make` to make the executable; adding `clean` to remove the produced object files, `fclean` to
get rid of everything except source files, and `re` to initiate a complete clean build.
